    • 使用了，于是乎将Clash关掉

  • 重新按照流程生成一遍ssh秘钥，然后填到github上（流程复杂，耗时
    • ssh-keygen -t rsa -b 4096 -C “xxx@163.com“(填自己的邮箱)
    • eval “$(ssh-agent -s)”
    • ssh-add ~/.ssh/id_rsa
    • 打开~/.ssh/rsa.pub，复制公钥到github
