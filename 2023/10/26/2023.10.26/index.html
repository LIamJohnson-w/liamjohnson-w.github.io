гҖҗRedisгҖ‘Redisеёёз”Ёе‘Ҫд»Ө

Created2023-10-26|Updated2023-11-02
|Post View:

иҜҙжҳҺпјҡиҝҷжҳҜдёҖдёӘдёҚеӨӘжҲҗзҶҹзҡ„еҚҡе®ўпјҢRedisз®—жҳҜеҲқж¬ЎжҺҘи§ҰпјҢеҫҲеӨҡдёңиҘҝйғҪдёҚдјҡпјҢи·ҜиҝҳеҫҲй•ҝпјҢеҫҲеӨҡдёңиҘҝиҰҒеӯҰгҖӮ

еҸӮиҖғпјҡ

  • Redisе‘Ҫд»ӨеӨ§е…ЁпјҲи¶…иҜҰз»ҶпјүеҚҡе®ўеӣӯ
  • Redis еёёз”Ёж“ҚдҪңе‘Ҫд»ӨпјҢйқһеёёиҜҰз»ҶпјҒ зҹҘд№Һ

еҗҜеҠЁе’Ңе…ій—ӯ

иҝңзЁӢиҝһжҺҘRedis

keyж“ҚдҪңе‘Ҫд»Ө

Redisжү№йҮҸеҲ йҷӨkeyеҖј

еҸӮиҖғпјҡredisеҰӮдҪ•дјҳйӣ…еҲ йҷӨеӨ§йҮҸзҡ„keyпјҹ

  • ж–№ејҸдёҖпјҡ
    • redis-cli -h ең°еқҖ -p 6379 -a еҜҶз Ғ keys вҖңеҢ№й…ҚеҶ…е®№вҖқ | xargs ./redis-cli -h ең°еқҖ -p 6379 -a еҜҶз Ғ del
  • ж–№ејҸдәҢпјҡпјҲз”ұдәҺж–№ејҸдёҖеҸҜиғҪдјҡеј•иө·CPUиө„жәҗзҡ„еӨ§йҮҸж¶ҲиҖ—пјҢеҸҜиғҪдјҡеҜ№еҲ«зҡ„жңҚеҠЎйҖ жҲҗжҢӨеҺӢпјү
    • redis-cli -h ең°еқҖ -a еҜҶз Ғ -n ж“ҚдҪңDB scan 0 match вҖңеҢ№й…ҚеҶ…е®№вҖқ count жү«жҸҸж•°йҮҸ | xargs -t redis-cli -h ең°еқҖ -a еҜҶз Ғ -n ж“ҚдҪңDB del
    • ејҠз«ҜпјҡжңүеҸҜиғҪжү«жҸҸдёҚе®Ңж•ҙдёӘеә“
  • жңҖз»Ҳи§ЈеҶіж–№жЎҲпјҡ
    • scanе‘Ҫд»Өдјҡиҝ”еӣһ2дёӘеҖјгҖӮз¬¬дёҖдёӘдёәжёёж Үз§»еҠЁеҗҺзҡ„ж–°дҪҚзҪ®пјҢз¬¬дәҢдёӘжҳҜеҢ№й…ҚеҲ°зҡ„keyеҖј
    • Shellи„ҡжң¬

еӯ—з¬ҰдёІж“ҚдҪңе‘Ҫд»Ө

еӯ—з¬ҰдёІжҳҜRedisдёӯжңҖеҹәжң¬зҡ„ж•°жҚ®зұ»еһӢпјҢеҚ•дёӘж•°жҚ®иғҪеӯҳеӮЁзҡ„жңҖеӨ§з©әй—ҙжҳҜ512MгҖӮ

еӯҳж”ҫй”®еҖј

иҜӯжі•пјҡset key value [EX seconds] [PX milliseconds] [NX|XX]

nxпјҡеҰӮжһңkeyдёҚеӯҳеңЁеҲҷе»әз«ӢпјҢxxпјҡеҰӮжһңkeyеӯҳеңЁеҲҷдҝ®ж”№е…¶еҖјпјҢд№ҹеҸҜд»ҘзӣҙжҺҘдҪҝз”Ёsetnx/setexе‘Ҫд»ӨгҖӮ

иҺ·еҸ–й”®еҖј

иҜӯжі•пјҡget key

еҖјйҖ’еўһ/йҖ’еҮҸ

еҰӮжһңеӯ—з¬ҰдёІдёӯзҡ„еҖјжҳҜж•°еӯ—зұ»еһӢзҡ„пјҢеҸҜд»ҘдҪҝз”Ёincrе‘Ҫд»ӨжҜҸж¬ЎйҖ’еўһпјҢдёҚжҳҜж•°еӯ—зұ»еһӢеҲҷжҠҘй”ҷгҖӮ

иҜӯжі•пјҡincr key

дёҖж¬ЎжғійҖ’еўһNз”Ёincrbyе‘Ҫд»ӨпјҢеҰӮжһңжҳҜжө®зӮ№еһӢж•°жҚ®еҸҜд»Ҙз”Ёincrbyfloatе‘Ҫд»ӨйҖ’еўһгҖӮ

еҗҢж ·пјҢйҖ’еҮҸдҪҝз”ЁdecrгҖҒdecrbyе‘Ҫд»ӨгҖӮ

жү№йҮҸеӯҳж”ҫй”®еҖј

иҜӯжі•пјҡmset key value [key value вҖҰ]

иҝһз»ӯиҺ·еҸ–й”®еҖј

иҜӯжі•пјҡmget key [key вҖҰ]

RedisжҺҘж”¶зҡ„жҳҜUTF-8зҡ„зј–з ҒпјҢеҰӮжһңжҳҜдёӯж–ҮдёҖдёӘжұүеӯ—е°ҶеҚ 3дҪҚиҝ”еӣһгҖӮ

иҺ·еҸ–еҖјй•ҝеәҰ

иҜӯжі•пјҡstrlen key
127.0.0.1:6379[2]> strlen javastack (integer) 3

иҝҪеҠ еҶ…е®№

иҜӯжі•пјҡappend key value

еҗ‘й”®еҖје°ҫйғЁж·»еҠ пјҢеҰӮдёҠе‘Ҫд»Өжү§иЎҢеҗҺз”ұ666еҸҳжҲҗ666hi

иҺ·еҸ–йғЁеҲҶеӯ—з¬Ұ

иҜӯжі•пјҡgetrange key start end

йӣҶеҗҲж“ҚдҪңе‘Ҫд»Ө

еӯҳеӮЁеҖј

иҜӯжі•пјҡsadd key member [member вҖҰ]

иҺ·еҸ–е…ғзҙ 

иҺ·еҸ–жүҖжңүе…ғзҙ иҜӯжі•пјҡsmembers key

йҡҸжңәиҺ·еҸ–иҜӯжі•пјҡsrandmember langs count

еҲӨж–ӯйӣҶеҗҲжҳҜеҗҰеӯҳеңЁе…ғзҙ 

иҜӯжі•пјҡsismember key member

иҺ·еҸ–йӣҶеҗҲе…ғзҙ дёӘж•°

иҜӯжі•пјҡscard key

еҲ йҷӨйӣҶеҗҲе…ғзҙ 

иҜӯжі•пјҡsrem key member [member вҖҰ]

еј№еҮәе…ғзҙ 

иҜӯжі•пјҡspop key [count]

жңүеәҸйӣҶеҗҲж“ҚдҪңе‘Ҫд»Ө

е’ҢеҲ—иЎЁзҡ„еҢәеҲ«пјҡ

1гҖҒеҲ—иЎЁдҪҝз”Ёй“ҫиЎЁе®һзҺ°пјҢдёӨеӨҙеҝ«пјҢдёӯй—ҙж…ўгҖӮжңүеәҸйӣҶеҗҲжҳҜж•ЈеҲ—иЎЁе’Ңи·іи·ғиЎЁе®һзҺ°зҡ„пјҢеҚідҪҝиҜ»еҸ–дёӯй—ҙзҡ„е…ғзҙ д№ҹжҜ”иҫғеҝ«гҖӮ

2гҖҒеҲ—иЎЁдёҚиғҪи°ғж•ҙе…ғзҙ дҪҚзҪ®пјҢжңүеәҸйӣҶеҗҲиғҪгҖӮ

3гҖҒжңүеәҸйӣҶеҗҲжҜ”еҲ—иЎЁжӣҙеҚ еҶ…еӯҳгҖӮ

еӯҳеӮЁеҖј

иҜӯжі•пјҡzadd key [NX|XX] [CH] [INCR] score member [score member вҖҰ]

иҺ·еҸ–е…ғзҙ еҲҶж•°

иҜӯжі•пјҡzscore key member

иҺ·еҸ–жҺ’еҗҚиҢғеӣҙжҺ’еҗҚиҜӯжі•пјҡzrange key start stop [WITHSCORES]

иҺ·еҸ–жҢҮе®ҡеҲҶж•°иҢғеӣҙжҺ’еҗҚиҜӯжі•пјҡzrangebyscore key min max [WITHSCORES] [LIMIT offset count]

еўһеҠ жҢҮе®ҡе…ғзҙ еҲҶж•°

иҜӯжі•пјҡzincrby key increment member

иҺ·еҸ–йӣҶеҗҲе…ғзҙ дёӘж•°

иҜӯжі•пјҡzcard key

иҺ·еҸ–жҢҮе®ҡиҢғеӣҙеҲҶж•°дёӘж•°

иҜӯжі•пјҡzcount key min max

еҲ йҷӨжҢҮе®ҡе…ғзҙ 

иҜӯжі•пјҡzrem key member [member вҖҰ]

иҺ·еҸ–е…ғзҙ жҺ’еҗҚ

иҜӯжі•пјҡzrank key member

еҲ—иЎЁж“ҚдҪңе‘Ҫд»Ө

еҲ—иЎЁзұ»еһӢжҳҜдёҖдёӘжңүеәҸзҡ„еӯ—ж®өдёІеҲ—иЎЁпјҢеҶ…йғЁжҳҜдҪҝз”ЁеҸҢеҗ‘й“ҫиЎЁе®һзҺ°пјҢжүҖжңүеҸҜд»Ҙеҗ‘дёӨз«Ҝж“ҚдҪңе…ғзҙ пјҢиҺ·еҸ–дёӨз«Ҝзҡ„ж•°жҚ®йҖҹеәҰеҝ«пјҢйҖҡиҝҮзҙўеј•еҲ°е…·дҪ“зҡ„иЎҢж•°жҜ”иҫғж…ўгҖӮ

еҲ—иЎЁзұ»еһӢзҡ„е…ғзҙ жҳҜжңүеәҸдё”еҸҜд»ҘйҮҚеӨҚзҡ„гҖӮ

еӯҳеӮЁеҖј

е·Ұз«ҜеӯҳеҖјиҜӯжі•пјҡlpush key value [value вҖҰ]

еҸіз«ҜеӯҳеҖјиҜӯжі•пјҡrpush key value [value вҖҰ]

зҙўеј•еӯҳеҖјиҜӯжі•пјҡlset key index value

еј№еҮәе…ғзҙ 

е·Ұз«Ҝеј№еҮәиҜӯжі•пјҡlpop key

еҸіз«Ҝеј№еҮәиҜӯжі•пјҡrpop key

иҺ·еҸ–е…ғзҙ дёӘж•°

иҜӯжі•пјҡllen key

иҺ·еҸ–еҲ—иЎЁе…ғзҙ 

дёӨиҫ№иҺ·еҸ–иҜӯжі•пјҡlrange key start stop

зҙўеј•иҺ·еҸ–иҜӯжі•пјҡlindex key index

еҲ йҷӨе…ғзҙ 

ж №жҚ®еҖјеҲ йҷӨиҜӯжі•пјҡlrem key count value

иҢғеӣҙеҲ йҷӨиҜӯжі•пјҡltrim key start stop

ж•ЈеҲ—ж“ҚдҪңе‘Ҫд»Ө

redisеӯ—з¬ҰдёІзұ»еһӢй”®е’ҢеҖјжҳҜеӯ—е…ёз»“жһ„еҪўејҸпјҢиҝҷйҮҢзҡ„ж•ЈеҲ—зұ»еһӢе…¶еҖјд№ҹеҸҜд»ҘжҳҜеӯ—е…ёз»“жһ„гҖӮ

еӯҳж”ҫй”®еҖј

еҚ•дёӘиҜӯжі•пјҡhset key field value
