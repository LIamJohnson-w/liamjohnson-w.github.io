жҠҪиұЎзұ»

жҠҪиұЎзұ»зҡ„з”ұжқҘ:

еӣһйЎҫжҳЁеӨ©еҶҷзҡ„еҠЁзү©зұ»д»Јз Ғ, жҲ‘д»¬еңЁе…¶дёӯеҶҷдәҶ1дёӘeat()ж–№жі•, ж–№жі•дёӯжңүдёҖеҸҘвҖқеәҹиҜқвҖқ: еҠЁзү©дјҡеҗғ.дёҚеҗҢзҡ„еҠЁзү©еҗғзҡ„еҶ…е®№жҳҜдёҚдёҖж ·зҡ„, жүҖд»ҘиҝҷеҸҘиҜқеҶҷдәҶд№ҹжІЎжңүж„Ҹд№ү, жҲ‘д»¬е°ұжҖқиҖғ, иғҪдёҚиғҪеҲ жҺүе‘ў?иҝҷж ·зҡ„ж–№жі•е°ұеҸ«: жҠҪиұЎж–№жі•, жңүжҠҪиұЎж–№жі•зҡ„зұ», е°ұеҸ«жҠҪиұЎзұ», Javaдёӯ, жҠҪиұЎз”ЁabstractиЎЁзӨә.

жҠҪиұЎзұ»зҡ„зү№зӮ№:

  1. Javaдёӯ, жҠҪиұЎз”Ё abstractе…ій”®еӯ—иЎЁзӨә.
  2. жңүжҠҪиұЎж–№жі•зҡ„зұ»дёҖе®ҡжҳҜжҠҪиұЎзұ»(жҲ–иҖ…жҺҘеҸЈ), дҪҶжҳҜжҠҪиұЎзұ»дёӯдёҚдёҖе®ҡжңүжҠҪиұЎж–№жі•.
  3. жҠҪиұЎзұ»дёҚиғҪзӣҙжҺҘе®һдҫӢеҢ–.
    й—®: йӮЈжҠҪиұЎзұ»жҖҺд№Ҳе®һдҫӢеҢ–е‘ў?
    зӯ”: йҖҡиҝҮ жҠҪиұЎзұ»еӨҡжҖҒ е®һзҺ°, еҚі: еҲӣе»әиҜҘжҠҪиұЎзұ»зҡ„еӯҗзұ»еҜ№иұЎеҚіеҸҜ.
  4. жҠҪиұЎзұ»зҡ„еӯҗзұ».
    еҰӮжһңжҳҜжҷ®йҖҡзұ»: еҲҷеҝ…йЎ»йҮҚеҶҷиҜҘжҠҪиұЎзұ»дёӯжүҖжңүзҡ„жҠҪиұЎж–№жі•.
    еҰӮжһңжҳҜжҠҪиұЎзұ»: еҲҷеҸҜд»ҘдёҚз”ЁйҮҚеҶҷзҲ¶зұ»дёӯзҡ„жҠҪиұЎж–№жі•.

жҠҪиұЎзұ»зҡ„жҲҗе‘ҳзү№зӮ№: (еҚі, жҠҪиұЎзұ»дёӯиғҪеҶҷе•Ҙ, дёҚиғҪеҶҷе•Ҙ.)

дё“дёҡзүҲ: еҸҳйҮҸ, еёёйҮҸ, жһ„йҖ ж–№жі•, йқһйқҷжҖҒж–№жі•, йқҷжҖҒж–№жі•, жҠҪиұЎж–№жі•.
з®ҖеҚ•и®°еҝҶ: жҠҪиұЎзұ»жҜ”жҷ®йҖҡзұ»еӨҡдёҖз§ҚжҠҪиұЎж–№жі•, иҖҢдё”иҝҳеҸҜд»ҘдёҚеҶҷ.

й—®: жҠҪиұЎзұ»дёӯзҡ„ жҠҪиұЎж–№жі• е’Ң йқһжҠҪиұЎж–№жі•жңүд»Җд№Ҳж„Ҹд№ү?

жҠҪиұЎж–№жі•: ејәеҲ¶иҰҒжұӮеӯҗзұ»еҝ…йЎ»е®ҢжҲҗжҹҗдәӣж“ҚдҪң.
йқһжҠҪиұЎж–№жі•: и®©еӯҗзұ»з»§жүҝ, жҸҗй«ҳд»Јз Ғзҡ„еӨҚз”ЁжҖ§.

еҝ«жҚ·й”®:

alt + enter(еӣһиҪҰ)

йқўеҗ‘еҜ№иұЎжҺҘеҸЈ

жҺҘеҸЈзҡ„з”ұжқҘ

жҲ‘д»¬зҹҘйҒ“, жҠҪиұЎзұ»дёӯдёҚд»…еҸҜд»ҘеҶҷжҠҪиұЎж–№жі•, иҝҳеҸҜд»ҘеҶҷеҸҳйҮҸ, еёёйҮҸ, жһ„йҖ ж–№жі•, йқҷжҖҒж–№жі•, йқһжҠҪиұЎж–№жі•зӯү, жүҖд»ҘжҠҪиұЎзұ»зӣёеҜ№ вҖңдёҚзәҜзІ№вҖқ,
иҖҢдё”, еӯҗзұ»еҸҜд»ҘйҖҡиҝҮз»§жүҝзҡ„ж–№ејҸд»ҺзҲ¶зұ»з»§жүҝ йҷӨзҲ¶жһ„йҖ ж–№жі•еӨ–, жүҖжңүзҡ„е…¬е…ұеҶ…е®№(publicдҝ®йҘ°зҡ„), иҝҷз§Қж–№ејҸ, иҖҰеҗҲжҖ§жҜ”иҫғејә, дәҺжҳҜ,
е°ұеҮәзҺ°дәҶжҺҘеҸЈ, е°ұжҳҜз”ЁжқҘ: йҷҚдҪҺиҖҰеҗҲжҖ§зҡ„.

дёҫдҫӢ

й—®: еҒҮи®ҫAnimal(еҠЁзү©зұ»), жңүеӯҗзұ»Cat(зҢ«зұ»), Dog(зӢ—зұ»), йғЁеҲҶзҡ„зҢ«, з»ҸиҝҮдәҶи®ӯз»ғ, еӯҰдјҡдәҶи·ій«ҳ, иҜ·й—®: и·ій«ҳзҡ„еҠҹиғҪеә”иҜҘеҶҷеҲ°е“ӘйҮҢ?
зӯ”:
еҶҷеҲ°Animalзұ»? иҝҷе°ұж„Ҹе‘ізқҖ жүҖжңүзҡ„ еҠЁзү© йғҪдјҡи·ій«ҳ, дёҚеҗҲйҖӮ.
еҶҷеҲ°Catзұ»? иҝҷе°ұж„Ҹе‘ізқҖ жүҖжңүзҡ„ зҢ« йғҪдјҡи·ій«ҳ, дёҚеҗҲйҖӮ.
еҶҷеҲ°JumpCatзұ»дёӯ? еҸҜд»Ҙ, дҪҶжҳҜдёҚжҺЁиҚҗ, дёҮдёҖе…¶е®ғзҡ„еҠЁзү©з»ҸиҝҮи®ӯз»ғ, д№ҹеӯҰдјҡдәҶи·ій«ҳе‘ў? дёҚж–№дҫҝйӣҶдёӯз®ЎзҗҶ.
жҺЁиҚҗ: е®ҡд№ү1дёӘжҺҘеҸЈ, йҮҢиҫ№жңүи·ій«ҳзҡ„еҠҹиғҪ, и°Ғдјҡи·ій«ҳ, и°Ғе®һзҺ°иҝҷдёӘжҺҘеҸЈеҚіеҸҜ.

жҺҘеҸЈзҡ„зү№зӮ№

  1. Javaдёӯ, жҺҘеҸЈз”Ё interfaceе…ій”®еӯ—иЎЁзӨә, зұ»дёҺжҺҘеҸЈд№Ӣй—ҙжҳҜе®һзҺ°е…ізі»,з”Ё implementsдҝ®йҘ°.
  2. жңүжҠҪиұЎж–№жі•зҡ„зұ»дёҖе®ҡжҳҜжҠҪиұЎзұ»(жҲ–иҖ…жҺҘеҸЈ), дҪҶжҳҜжҠҪиұЎзұ»(жҺҘеҸЈ)дёӯдёҚдёҖе®ҡжңүжҠҪиұЎж–№жі•.
  3. жҺҘеҸЈдёҚиғҪзӣҙжҺҘе®һдҫӢеҢ–.
    й—®: йӮЈжҺҘеҸЈжҖҺд№Ҳе®һдҫӢеҢ–е‘ў?
    зӯ”: йҖҡиҝҮ жҺҘеҸЈеӨҡжҖҒ е®һзҺ°, еҚі: еҲӣе»әиҜҘжҺҘеҸЈзҡ„еӯҗзұ»еҜ№иұЎеҚіеҸҜ.
  4. жҺҘеҸЈзҡ„еӯҗзұ».
    еҰӮжһңжҳҜжҷ®йҖҡзұ»: еҲҷеҝ…йЎ»йҮҚеҶҷиҜҘжҺҘеҸЈдёӯжүҖжңүзҡ„жҠҪиұЎж–№жі•.
    еҰӮжһңжҳҜжҠҪиұЎзұ»: еҲҷеҸҜд»ҘдёҚз”ЁйҮҚеҶҷзҲ¶жҺҘеҸЈдёӯзҡ„жҠҪиұЎж–№жі•.

жҺҘеҸЈзҡ„жҲҗе‘ҳзү№зӮ№

JDK1.5еҸҠе…¶д»ҘеүҚ: жҺҘеҸЈдёӯжңүдё”еҸӘиғҪжңү еёёйҮҸ, жҠҪиұЎж–№жі•.
еӣ дёәдҪ еҶҷзҡ„еҸҳйҮҸ, й»ҳи®Өдјҡжңү: public static final дҝ®йҘ°.
еӣ дёәдҪ еҶҷзҡ„ж–№жі•, й»ҳи®Өдјҡжңү: public abstract дҝ®йҘ°.
JDK1.8еҸҠе…¶д»ҘеҗҺ: жҺҘеҸЈдёӯеҸҜд»ҘеҶҷ йқҷжҖҒж–№жі• е’Ң й»ҳи®Өж–№жі•дәҶ.
йқҷжҖҒж–№жі•: иҰҒз”Ёstaticдҝ®йҘ°.
й»ҳи®Өж–№жі•: иҰҒз”Ёdefaultдҝ®йҘ°, е…¶е®һе°ұжҳҜжҲ‘д»¬еҶҷзҡ„йқһйқҷжҖҒж–№жі•.

жҺҘеҸЈ е’Ң (жҠҪиұЎ)зұ»зҡ„еҢәеҲ«

  1. жҲҗе‘ҳзү№зӮ№еҢәеҲ«.
    жҺҘеҸЈ: жңүдё”еҸӘиғҪжңүеёёйҮҸ, жҠҪиұЎж–№жі•, JDK1.8ејҖе§Ӣ, еҸҜд»ҘжңүйқҷжҖҒж–№жі• е’Ң й»ҳи®Өж–№жі•.
    зұ»: еҸҳйҮҸ, еёёйҮҸ, жһ„йҖ ж–№жі•, жҠҪиұЎж–№жі•, йқҷжҖҒж–№жі•, йқһйқҷжҖҒж–№жі•.
  2. е…ізі»зү№зӮ№еҢәеҲ«.
    зұ»дёҺзұ»: з»§жүҝе…ізі», еҸӘиғҪеҚ•з»§жүҝ, дёҚиғҪеӨҡз»§жүҝ, дҪҶжҳҜеҸҜд»ҘеӨҡеұӮз»§жүҝ.
    зұ»дёҺжҺҘеҸЈ: е®һзҺ°е…ізі», ж—ўеҸҜд»ҘеҚ•е®һзҺ°, д№ҹеҸҜд»ҘеӨҡе®һзҺ°, иҝҳеҸҜд»ҘеңЁз»§жүҝ1дёӘзұ»зҡ„еҗҢж—¶, е®һзҺ°еӨҡдёӘжҺҘеҸЈ.
    жҺҘеҸЈдёҺжҺҘеҸЈ: з»§жүҝе…ізі», еҸҜд»ҘеҚ•з»§жүҝ, д№ҹеҸҜд»ҘеӨҡз»§жүҝ.
  3. и®ҫи®ЎзҗҶеҝөзҡ„еҢәеҲ«.
    жҠҪиұЎзұ»: дёҖиҲ¬е……еҪ“зҲ¶зұ», е®ҡд№үзҡ„жҳҜж•ҙдёӘз»§жүҝдҪ“зі»зҡ„ е…ұжҖ§еҶ…е®№.
    жҺҘеҸЈ: дёҖиҲ¬е……еҪ“зҲ¶жҺҘеҸЈ, е®ҡд№үзҡ„жҳҜж•ҙдёӘз»§жүҝдҪ“зі»зҡ„ жү©еұ•еҶ…е®№.

еҶ…йғЁзұ»

еҶ…йғЁзұ»д»Ӣз»Қ:

еҶ…йғЁзұ»зҡ„жң¬иҙЁ: еҢҝеҗҚеҶ…йғЁзұ»жҳҜ1дёӘ з»§жүҝдәҶиҜҘзұ», жҲ–иҖ…е®һзҺ°дәҶиҜҘжҺҘеҸЈзҡ„ еҢҝеҗҚзҡ„ еӯҗзұ»еҜ№иұЎ.

жҰӮиҝ°:

еҶ…йғЁзұ»жҢҮзҡ„жҳҜзұ»йҮҢиҫ№иҝҳжңү1дёӘзұ», йҮҢиҫ№зҡ„зұ»еҸ«: еҶ…йғЁзұ», еӨ–иҫ№зҡ„зұ»еҸ«: еӨ–йғЁзұ».

еҲҶзұ»:

ж №жҚ®еҶ…йғЁзұ»зҡ„е®ҡд№үдҪҚзҪ®дёҚеҗҢ, еҶ…йғЁзұ»еҸҜд»ҘеҲҶдёә:
жҲҗе‘ҳеҶ…йғЁзұ»: дёҖиҲ¬жҳҜеә•еұӮжәҗз Ғзҡ„ж—¶еҖҷдјҡз”Ё, иЎЁзӨәеҜ№иҜҘзұ»зҡ„еҠҹиғҪеҒҡ еҠ ејә, жү©еұ•е»¶дјё. дҫӢеҰӮ: ArrayList
еұҖйғЁеҶ…йғЁзұ»: дёҖиҲ¬з”ЁдәҺеҝ«йҖҹзҡ„еҲқе§ӢеҢ–жҹҗдёӘжҠҪиұЎзұ»(жҺҘеҸЈ), еҚі: еҲӣе»әе®ғзҡ„еӯҗзұ»еҜ№иұЎ.

еҢҝеҗҚеҶ…йғЁзұ»:

жҰӮиҝ°:
жІЎжңүеҗҚеӯ—зҡ„ еұҖйғЁеҶ…йғЁзұ».
ж јејҸ:
new зұ»еҗҚжҲ–иҖ…жҺҘеҸЈеҗҚ() {
//йҮҚеҶҷзұ»жҲ–иҖ…жҺҘеҸЈзҡ„жүҖжңүжҠҪиұЎж–№жі•
};

жң¬иҙЁ: йҮҚзӮ№.

вҖӢ еҢҝеҗҚеҶ…йғЁзұ»жҳҜ1дёӘ з»§жүҝдәҶиҜҘзұ», жҲ–иҖ…е®һзҺ°дәҶиҜҘжҺҘеҸЈзҡ„ еҢҝеҗҚзҡ„ еӯҗзұ»еҜ№иұЎ.

еә”з”ЁеңәжҷҜ:

   1. еҢҝеҗҚеҶ…йғЁзұ»еҸҜд»ҘдҪңдёә ж–№жі•зҡ„е®һеҸӮ иҝӣиЎҢдј йҖ’.
   2. еҪ“еҜ№жҲҗе‘ҳж–№жі•д»…и°ғз”ЁдёҖж¬Ўзҡ„ж—¶еҖҷ.

APIз®Җд»Ӣ

APIе…Ёз§°жҳҜApplication Programming Interface(еә”з”ЁзЁӢеәҸзј–зЁӢжҺҘеҸЈ), жң¬ж„ҸжҢҮзҡ„жҳҜJDKжҸҗдҫӣзҡ„еҗ„з§ҚеҠҹиғҪзҡ„Javaзұ»е’ҢжҺҘеҸЈ, дҪҶжҳҜжҲ‘д»¬еёёиҜҙзҡ„вҖқжү“ејҖAPIвҖқе№¶дёҚжҳҜжҢҮжү“ејҖиҝҷдәӣJavaзұ»е’ҢжҺҘеҸЈ, иҖҢжҳҜжү“ејҖAPIеё®еҠ©ж–ҮжЎЈ.

API её®еҠ©ж–ҮжЎЈе°ұзӣёеҪ“дәҺиҜҙжҳҺд№Ұ,жҳҜз”ЁжқҘд»Ӣз»ҚJDKжҸҗдҫӣзҡ„еҗ„дёӘзұ»е’ҢжҺҘеҸЈзҡ„еҠҹиғҪзҡ„, д»ҺиҖҢеё®еҠ©жҲ‘д»¬еҝ«йҖҹдёҠжүӢдҪҝз”Ё.

APIеё®еҠ©ж–ҮжЎЈзҡ„дҪҝз”ЁжӯҘйӘӨ:
1. жүҫеҲ°дҪ иҰҒз”Ёзҡ„зұ».
2. зңӢзұ»зҡ„з»“жһ„.
еҚі: зҲ¶зұ»жҳҜи°Ғ, жңүе“Әдәӣеӯҗзұ», еңЁе“ӘдёӘеҢ…дёӢзӯүвҖҰ
еҰӮжһң java.lang еҢ…дёӢзҡ„зұ», еҸҜд»ҘзӣҙжҺҘдҪҝз”Ё, е…¶е®ғеҢ…дёӢзҡ„зұ», з”Ёд№ӢеүҚйңҖиҰҒеҜјеҢ….
3. еӨ§жҰӮзҡ„зңӢдёӢзұ»зҡ„иҜҙжҳҺ, зҹҘйҒ“иҜҘзұ»жҳҜе№Іе•Ҙзҡ„.
4. зңӢжһ„йҖ ж–№жі•.
жңүжһ„йҖ : иҜҙжҳҺж–№жі•еҹәжң¬дёҠйғҪжҳҜ йқһйқҷжҖҒзҡ„, йңҖиҰҒйҖҡиҝҮ еҜ№иұЎеҗҚ. зҡ„ж–№ејҸи°ғз”Ё.
ж— жһ„йҖ : иҜҙжҳҺж–№жі•еҹәжң¬дёҠйғҪжҳҜ йқҷжҖҒзҡ„, еҸҜд»ҘйҖҡиҝҮ зұ»еҗҚ. зҡ„ж–№ејҸи°ғз”Ё.
5. е…·дҪ“зҡ„и°ғз”Ёж–№жі•зҡ„иҝҮзЁӢ.
жіЁж„Ҹж–№жі•еҗҚеӨ§е°ҸеҶҷ, еҲ«еҶҷй”ҷдәҶ, дј еҸӮ, жҺҘж”¶иҝ”еӣһеҖјзӯүвҖҰ

дә§з”ҹеҺҹеӣ :

   1. nextInt() е’Ң nextLine()зҡ„з»“жқҹж Үи®°йғҪжҳҜ: \r\n
   2. nextInt() еҸӘжҺҘж”¶з”ЁжҲ·еҪ•е…Ҙзҡ„ж•°еӯ—, дёҚжҺҘж”¶жңҖеҗҺзҡ„ еӣһиҪҰжҚўиЎҢз¬Ұ(\r\n)
   3. nextInt()"йҒ—з•ҷ"дёӢжқҘзҡ„ \r\n, иў«nextLine()зӣҙжҺҘиҜҶеҲ«дәҶ, еҜјиҮҙnextLine()зӣҙжҺҘз»“жқҹ.

е®ғжҳҜжүҖжңүзұ»зҡ„зҲ¶зұ», д»»дҪ•зҡ„зұ»йғҪзӣҙжҺҘжҲ–иҖ…й—ҙжҺҘз»§жүҝиҮӘObject, Objectзұ»жҳҜжүҖжңүзұ»зҡ„зҲ¶зұ», еҹәзұ», и¶…зұ».

public String toString(); й»ҳи®Өжү“еҚ°зҡ„жҳҜеҜ№иұЎзҡ„ең°еқҖеҖј, ж— ж„Ҹд№ү, дёҖиҲ¬дјҡйҮҚеҶҷе®ғ, ж”№дёәжү“еҚ°иҜҘеҜ№иұЎзҡ„еҗ„дёӘеұһжҖ§еҖј.

з»ҶиҠӮ:

1. иҫ“еҮәиҜӯеҸҘзӣҙжҺҘжү“еҚ°еҜ№иұЎ, й»ҳи®Өи°ғз”ЁдәҶиҜҘеҜ№иұЎзҡ„ toString()ж–№жі•.
